# Turbid Portfolio - Obfuscated Version

## 🛡️ Protection Features

This directory contains heavily obfuscated versions of your portfolio files designed to make code difficult to understand and copy while maintaining full functionality.

### 🔒 Security Measures Implemented:

1. **Advanced Anti-Debug Protection**
   - Developer tools detection
   - Console access blocking
   - Debugger statement detection
   - Automatic page closure on suspicious activity

2. **Content Protection**
   - Right-click disabled
   - Text selection limited
   - Copy/paste functions blocked
   - Drag operations prevented
   - Clipboard clearing every 5 seconds

3. **Keyboard Shortcuts Blocked**
   - F12 (Developer Tools)
   - Ctrl+Shift+I/J/C
   - Ctrl+U/S
   - Meta+Alt+I/J/C (Mac)

4. **Iframe Protection**
   - Prevents embedding in iframes
   - Automatic redirect if detected

### 📁 Files Structure

- **index.html** - Obfuscated HTML with embedded protection scripts
- **style.css** - Minified and obfuscated CSS
- **script.js** - Heavily obfuscated JavaScript with Base64 encoding
- **README.md** - This documentation file

### 🔐 Obfuscation Techniques Used

1. **Base64 Encoding** - String content encoded in Base64
2. **Variable Obfuscation** - Random variable names (e.g., `_0x1a2b`, `_0x3c4d`)
3. **Function Obfuscation** - Function names encoded and decoded at runtime
4. **Dynamic Content Loading** - Content loaded via JavaScript encoding
5. **Anti-Tampering** - Multiple layers of protection against code inspection
